Markdown Support

The comment widget supports basic Markdown formatting:

  • Bold text: Use **text** or __text__
  • Italic text: Use *text* or _text_
  • Links: Use [text](url)
  • Inline code: Use `code`
  • Code blocks
    : Use ```code```
  • Headers: Use # Header, ## Header, etc.
