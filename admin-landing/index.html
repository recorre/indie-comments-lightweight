Get Started

Integrate Indie Comments into your site in just a few steps:

1. Include the widget

<!-- Include the widget component -->
<script src="https://yourdomain.com/comment-widget.js"></script>

2. Add the widget to your page

<!-- Add this where you want comments to appear -->
<comment-widget thread-id="your-unique-id" page-title="Page Title"></comment-widget>

3. Configure your backend

Set up your backend proxy with your API keys (never exposed to frontend).
